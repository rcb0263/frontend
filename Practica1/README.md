Usage
Make sure to install Deno: https://deno.land/manual/getting_started/installation

Then start the project:

deno task start

This will watch the project directory and restart as necessary.

once started, use routes: /search  or /starships

Ignorar /character y /characters, en el caso de fails y starshipfail, son intentos fallidos, no hacen todo lo que deben o no funcionan
